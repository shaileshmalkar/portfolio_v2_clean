# Resume Folder

## How to Add Your Resume

1. **Place your resume PDF here** in this folder (`frontend/public/resume/`)
   - Name it: `resume.pdf` (or update the filename in the Resume component)

2. **Supported formats**: PDF (recommended)

3. **The resume will be:**
   - Displayed in an embedded PDF viewer on the Resume page
   - Available for download via the download button

## Example Structure:
```
frontend/public/resume/
  └── resume.pdf
```

## Note:
- Make sure your resume PDF is optimized for web viewing
- Keep file size reasonable for faster loading
- The resume will be accessible at `/resume/resume.pdf`
